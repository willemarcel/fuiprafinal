Sobre

Fui pra final! é um aplicativo que calcula a nota necessária para o estudante ser aprovado na Prova Final. Inicialmente foi feito para a UFRB, mas queremos fazer versões para outras universidades.

O aplicativo foi criado por César Velame e Wille Marcel

Esse aplicativo é Software Livre, licenciado sob a licença GNU GPLv2. O código fonte está disponível no GitHub. Fork us!

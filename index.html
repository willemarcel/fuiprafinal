Fui pra Final! é um aplicativo que calcula a nota necessária na prova final para o estudante ser aprovado. Inicialmente foi feito para a UFRB, mas a intenção é desenvolver versões para outras Universidades.

Esse aplicativo é Software Livre, licenciado sob a licença GNU GPLv2. O código fonte está disponível no GitHub. Fork us!

Desenvolvido por César Velame e Wille Marcel.
